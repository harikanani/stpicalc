# stpicalc

A Simple NPM Package to calculate STPI

## Installation

Install stpicalc with npm

```bash
  npm i stpicalc --save
```

then...

```typescript
  import { stpicalc } from "stpicalc";

  let stpi = stpicalc([10, 5, 10, 7], [7, 7, 7, 7], [10, 10, 10, 10], [6, 7, 7, 7]);
```

## Tech Stack

-   Typescript
